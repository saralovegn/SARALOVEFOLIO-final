# Personal Portfolio Web

Este repositorio contiene mi portfolio personal, basado en un template proporcionado para la clase de Programación Web.

---

## Descripción del Proyecto

El portfolio se centra en una experiencia visual con modo oscuro, animaciones y secciones dedicadas a cada proyecto. La página está completamente en inglés.

---

## Cambios y Personalizaciones Realizadas

En el primer commit, realicé las siguientes modificaciones:

- **Modo oscuro**: Mantuve el modo oscuro del template original, pero ajusté ligeramente los tonos para mejorar la legibilidad y estética general.
- **Idioma**: Cambié todo el contenido al inglés para adecuarlo a un público internacional.
- **Sección Hero**: Eliminé texto y mantuve solo imágenes en bucle o un vídeo de fondo para un impacto visual más fuerte.
- **Secciones de proyectos**: Cada proyecto tiene ahora su propia página individual, con más imágenes y una descripción detallada de cada uno para que los proyectos sean el centro de atención.
- **Animaciones y efectos visuales**:
  - Botones con efecto “shine” para hover.
  - Tarjetas de proyectos con hover flotante.
- **Footer**: Añadí mis enlaces o url personales hacia mi Instagram y GitHub.
